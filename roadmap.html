Targets

  • Beta 1

    • Complex, Matrix and Vector Classes functioning

  • 1.0 Release

    • Beta 1 thouroughly tested
    • Documentation for Beta 1 features complete
    • All methods aliased for ease of use in both scala and java

  • 1.X/2.X Releases

    These features will increment the 1.X number if they do not cause any backwards compatibility problems (i.e. past code will still compile against the library), otherwise they will be moved to the 2.X (and above) numbering.

    • Optimizations in algorithms (e.g. determinant).
    • Optimization in threading (e.g. matrix multiplication)
    • Imaginary numbers being own class?
    • Hypercomplex numbers (quaternions and above)
