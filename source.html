Getting the source

You my browse the source online at the Sedenion github page.

You can download the source code in either zip or tar format:

zip

tar

Alternitavely, you can download the source using Git by running: $ git clone git://github.com/Mithos/Sedenion

Compiling the source

Requiements

  • Java Developement Kit (JDK)
  • sbt

Building

  1. Extract the source
  2. Open a terminal/command prompt and cd to the directory containing the source
  3. Run $ sbt compile package doc
  4. Your finished jar and documentation will be in the new target/ folder
